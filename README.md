# Vizinho Seguro

O Vizinho Seguro é um MVP acadêmico de uma plataforma colaborativa de segurança comunitária.

A proposta consiste em permitir que moradores registrem ocorrências, visualizem alertas e acompanhem eventos de segurança em sua região.

## Funcionalidades do MVP

- Login simulado
- Cadastro simulado
- Registro de ocorrência
- Visualização de ocorrências recentes
- Mapa ilustrativo da comunidade
- Histórico de alertas
- Perfil do usuário

## Tecnologia Utilizada

- HTML
- CSS
- JavaScript
